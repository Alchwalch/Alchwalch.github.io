## Seq2Seq

기존 LSTMlm으로 언어를 다루기에는 문장하나에서 우러나오는 문맥을 활용하기 힘들다. 앞에 나온 입력된 단어만으로 결과를 출력하기에는 직관적으로도 부합하지 않는다. 그래서 이를 다룬 모델이 Seq2Seq구조이다. 

![seq2seq1](assets/img/seq2seq1.png)

위 그림과 같이 입력을 다 받아서 LSTM계열에서 context vector(h)를 내보내고 decoder는 그걸 받는다. 학습시에는 디코더가 주어진 데이터를 이용해 구하고 생성할때는 decoder에서 나온 단어를 다시 집어넣고 실행시키는 방식이다.

그러나 이렇게 모델을 만들면 성능이 좋지는 않다. 그래서 입력 데이터를 뒤집어서 처음 나오는 데이터들의 정확도를 올려주는 방법을 쓰는 방법도 있고 더 실용적인 방법은 **Peeky** 방법이 있다.

![seq2seq2](assets/img/seq2seq2.png)

Peeky 방법은 위 그림처럼 context vector를 앞에만 주는게 아니라 매 timestep 마다 concatenate시켜서 주는 방식이다. 그러면 성능이 많이 좋아진다.

구현을 해보자 들어오는 매 iteration마다 들어오는 token수는 고정이며 만약에 실제 내용을 담지 못하게 된다면 그 부분은 공백을 의미하는 토큰으로 매꾼다.
Encoder부분에서 마지막 timestep의 결과 h를 출력한다. 하지만 아래 코드에서는 모든 timestep의 h를 return하게 했다. attention에서는 모든 timestep의 context를 이용하기 때문에 이렇게 두었다.

```python
class Encoder(nn.Module):
  def __init__(self, input_size, hidden_size):
    super(Encoder, self).__init__()
    self.embedding = nn.Embedding(input_size, hidden_size)
    self.lstm = TimeLSTM(hidden_size, hidden_size)

  def forward(self, xs):
    xs = self.embedding(xs)
    return self.lstm(xs)
```

peeky방법을 이용하여 decoder를 만들었다. 아래 생략한 함수도 있는데 [코드](https://github.com/Alchwalch/Deep-Learning-Study/blob/main/NLP/seq2seq_attention.ipynb)를 보면 되겠다.

```python
class Seq2Seq_Decoder(nn.Module):
  def __init__(self, hidden_size, output_size,dropout=0.0):
    super(Seq2Seq_Decoder, self).__init__()
    self.embedding = nn.Embedding(output_size, hidden_size)
    self.lstm = TimeLSTM(hidden_size*2, hidden_size)
    self.linear = TimeLinear(hidden_size*2, output_size)
    self.dropout1=TimeDropout(dropout)
    self.dropout2=TimeDropout(dropout)

  def forward(self, xs, h):
    N, T = xs.shape
    xs = self.embedding(xs)
    xs = self.dropout1(xs)


    peeky_h = h.unsqueeze(1).expand(N, T, -1)
    xs = torch.cat([xs, peeky_h], dim=2)
    hs = self.lstm(xs, h)


    hs = self.dropout2(hs)
    hs = torch.cat([hs, peeky_h], dim=2)
    return self.linear(hs)
```

그리고 이를 연결해 주는 객체이다.

```python
class Seq2SeqPeeky(nn.Module):
  def __init__(self, vocab_size, hidden_size, dropout=0.0):
    super(Seq2SeqPeeky, self).__init__()
    self.encoder = Encoder(vocab_size, hidden_size)
    self.decoder = Seq2Seq_Decoder(hidden_size,vocab_size,dropout)
    self.softmax = TimeCrossEntropy()

  def forward(self, xs, ts):
    decoder_xs, targets = ts[:,:-1], ts[:,1:]

    self.encoder.lstm.reset_state()  # 배치마다 인코더 상태 초기화
    encoder_hs = self.encoder(xs)

    h = encoder_hs[:, -1, :]

    decoder_out = self.decoder(decoder_xs, h)
    return self.softmax(decoder_out, targets)
```

## Attention

위에서 보았다 시피 Seq2Seq는 한계가 많은 모델이다. Peeky같이 계산 비용도 들고 불안정한 기법을 써야 겨우겨우 제대로 된 성능을 냈다고 볼 수 있다. 그리고 더 문제는 입력이 길면 길어질 수록 한정된 벡터 안에 다 담아야 된다는 것이다.

Attention은 seq2seq에서 Decoder에 추가되는 layer인데 기존에 마지막 timestep의 h만 받는것과 달리 attention층에서는 모든 timestep에서 만든 h를 사용한다.

![seq2seq3](assets/img/seq2seq3.png)

각 단어의 벡터 hs는 decoder뒤에 쓰이는 attention layer에서 쓰인다. 이 벡터들은 attention 계층에서 decoder의 lstm을 기반으로 한 미리 계산된 **가중치 a**와 결합하여 아래 그림과 같이 결과를 출력한다.

![seq2seq4](assets/img/seq2seq4.png)

단어 벡타마다 확률을 나타내는 a를 구해보자. Decoder의 LSTM에서 도출된 h와 encoder에서 뽑은 hs를가지고 내적 시켜서 결과 s를 내보낸다. 그리고 그 s를 가지고 softmax를 돌리면 각 단어 벡터의 가중치이지 확률인 'a'가 만들어진다. 

![seq2seq5](assets/img/seq2seq5.png)

각각을 Weightsum레이어, AttentionWeight레이어라고 하며 AttentionWeight에서 encoder의 hs와 decoder의 lstm의 h를 받아 가중치 'a'를 도출하고 WeightSum에서 가중치와 hs를 받고 마지막 결과인 c를 출력하는 구조이다. 이를 다 합쳐서 attention 계층이라고 할 수 있다. 아래와 같이 간단하게 구현할 수 있다.

```python
class AttentionWeight(nn.Module):
  def __init__(self):
    super(AttentionWeight, self).__init__()
    self.softmax=nn.Softmax(dim=1)

  def forward(self,hs,h):
    s=torch.sum(hs*h.unsqueeze(1),dim=2)
    return self.softmax(s)
```

```python
class WeightSum(nn.Module):
  def __init__(self):
    super(WeightSum, self).__init__()

  def forward(self,hs,w):
    return torch.sum(hs*w.unsqueeze(2),dim=1)
```

```python
class Attention(nn.Module):
  def __init__(self):
    super(Attention, self).__init__()
    self.weight=AttentionWeight()
    self.weightSum=WeightSum()

  def forward(self,hs,h):
    w=self.weight(hs,h)
    return self.weightSum(hs,w),w
```

이렇게 Attention layer를 구현할 수 있으며 LSTM하고 Linear사이에 붙여서 사용한다. 디테일한건 [코드](https://github.com/Alchwalch/Deep-Learning-Study/blob/main/NLP/seq2seq_attention.ipynb)참고하자.
